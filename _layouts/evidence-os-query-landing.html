This namespace carries SHACL result-structure shapes only. The page below is a hand-written static landing per ADR-058 §2.11; pyLODE rendering is reserved for ontologies. The canonical artefact is the raw Turtle file linked above.

Scope

SHACL shapes that validate the structure of SPARQL query results produced by the evidence-os layer (KPI, trust-coverage, freshness, traceability, latency, change-link). These shapes do not declare an OWL ontology; they declare conformance contracts on result rows.
